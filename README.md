# Cash Register

### Test

To run the tests:

```console
$ npm run test
```

### Development

To start development, run in the console:
```console
$ npm start
``` 

Then open a browser and open the URL: http://localhost:3000

### JSDoc

To see JS documentation (JSDoc3)

```console
$ npm run jsdoc
```

Then open a browser and open the URL: http://localhost:3000/out/index.html
